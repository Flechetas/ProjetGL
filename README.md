# Projet Génie Logiciel - Réseau de neurones

## Dépendances

- SDL2 :
```bash
sudo apt install libsdl2-dev
```

## Build and run

```bash
make
./build/main
```
## Tests

```bash
make tests
```
## Docs

```bash
make docs
```
Entrypoint is docs/index.html

## Clean

```bash
make clean
```
