# Projet Génie Logiciel - Réseau de neurones

## Usage

### Build

```bash
make
```

### CLI

```bash
./build/main help
```

```bash
./build/main generate --help
```

```bash
./build/main train --help
```

```bash
./build/main draw --help
```

### Dependencies

- SDL2 :
```bash
sudo apt install libsdl2-dev
```

### Tests

```bash
make tests
```
### Docs

```bash
make docs
```
Entrypoint is docs/index.html

### Clean

```bash
make clean
```
